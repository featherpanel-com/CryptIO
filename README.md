# CryptIO Addon for FeatherPanel

A powerful and essential FeatherPanel addon that provides seamless encryption and decryption capabilities for strings using the same cryptographic algorithms as MythicalPanel.

## Features

- **Secure Encryption**: Encrypt sensitive strings using FeatherPanel's native encryption methods
- **Reliable Decryption**: Decrypt encrypted data with full compatibility
- **Database Integration**: Perfect for handling encrypted database credentials and sensitive configuration data
- **Command Line Interface**: Easy-to-use CLI commands for quick encryption/decryption operations

## Use Cases

- Decrypt encrypted database credentials stored in FeatherPanel
- Encrypt sensitive configuration values before storage
- Migrate encrypted data between environments
- Debug and troubleshoot encrypted data issues

This addon is particularly useful for administrators who need to work with FeatherPanel's encrypted database fields and configuration values.
